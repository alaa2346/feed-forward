# Simple ANN Implementation - Tanh Activation

This project is a Python implementation of a simple Artificial Neural Network (ANN) that performs a forward pass. It was developed as a solution for a neural networks course exercise.

## Features
- **Activation Function:** Uses `tanh` (Hyperbolic Tangent).
- **Weight Initialization:** Random weights in the range `[-0.5, 0.5]`.
- **Architecture:** - Input layer.
  - Hidden layer with bias `b1 = 0.5`.
  - Output layer with bias `b2 = 0.7`.

## Technologies
- Python
- NumPy
- Jupyter Notebook (`.ipynb`)

## How to use
Simply open the `.ipynb` file in Google Colab or Jupyter Notebook and run the cells to see the network output.
